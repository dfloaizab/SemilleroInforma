# Blender Cheat Sheet

## Navegación

| Acción | Atajo |
|---|---|
| Orbitar | MMB |
| Pan | Shift+MMB |
| Zoom | Rueda del mouse |
| Vista frontal | Numpad 1 |
| Vista lateral | Numpad 3 |
| Vista superior | Numpad 7 |
| Perspectiva/Ortográfica | Numpad 5 |
| Centrar objeto | Numpad . |

## Selección

| Acción | Atajo |
|---|---|
| Seleccionar | Click izquierdo |
| Selección múltiple | Shift+Click |
| Seleccionar todo | A |
| Invertir selección | Ctrl+I |
| Caja | B |
| Círculo | C |

## Transformaciones

| Acción | Atajo |
|---|---|
| Mover | G |
| Rotar | R |
| Escalar | S |
| Restringir eje | X Y Z |

## Modos

Tab cambia entre Object Mode y Edit Mode.

1: Vértices.

2: Aristas.

3: Caras.

## Modelado

E Extrude.

I Inset.

Ctrl+B Bevel.

Ctrl+R Loop Cut.

K Knife.

M Merge.

P Separate.

Shift+D Duplicar.

Ctrl+J Unir objetos.

X Eliminar.

## Modificadores

Boolean, Mirror, Array, Solidify, Bevel, Subdivision Surface y Decimate.

## Archivo

Ctrl+N Nuevo.

Ctrl+O Abrir.

Ctrl+S Guardar.

Ctrl+Shift+S Guardar como.

## Render

F12 Render.

Ctrl+F12 Animación.

## Generales

Ctrl+Z Deshacer.

Ctrl+Shift+Z Rehacer.

F3 Buscar comando.

H Ocultar.

Alt+H Mostrar.
