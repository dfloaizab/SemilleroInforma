# Sesión 3
# Transformación de Datos e Introducción a la Visualización

Este documento contiene la guía de la sesión sobre transformación de datos utilizando el caso de estudio de la tienda de mascotas.

## Objetivo

Al finalizar la sesión el estudiante será capaz de seleccionar información de un DataFrame, aplicar filtros, realizar agrupaciones, crear nuevas variables y construir sus primeras visualizaciones con Matplotlib.

## Flujo

Carga → Limpieza → Transformación → Visualización → Análisis.

## Contenido

- Acceso a columnas, filas y celdas (`loc` e `iloc`).
- Filtrado de registros.
- Ordenamiento.
- Agrupación con `groupby()`.
- Creación de nuevas variables.
- Exportación del DataFrame transformado.
- Introducción a gráficos de barras, histogramas y líneas con Matplotlib.

## Actividades

1. Consultar valores individuales del DataFrame.
2. Filtrar ventas por especie, barrio, estrato y producto.
3. Calcular promedios y totales mediante `groupby()`.
4. Crear columnas como IVA, ValorFinal y TipoCompra.
5. Generar gráficos de barras e histogramas.
6. Elaborar tres conclusiones a partir de las visualizaciones.

Este README acompaña el desarrollo práctico de la sesión y debe utilizarse junto con el archivo `tienda_mascotas_dataset_limpio.csv`.
