# Caso de estudio: Introducción al análisis de datos en una tienda de mascotas

Una tienda de mascotas desea iniciar un proceso de analítica de datos
para comprender mejor el comportamiento de sus ventas y apoyar futuras
decisiones comerciales. Para ello ha consolidado un registro de
transacciones que incluye la fecha de la venta, la especie y la raza de
la mascota, el barrio de residencia del cliente, el estrato
socioeconómico, el producto adquirido, la cantidad comprada, el precio
unitario y el valor total de cada compra.

El propósito de este caso de estudio es aplicar las primeras etapas del
flujo de trabajo en ciencia de datos. Inicialmente se espera que los
estudiantes carguen el archivo CSV utilizando Pandas, inspeccionen la
estructura del conjunto de datos, identifiquen los tipos de variables y
obtengan estadísticas descriptivas básicas mediante las funciones
head(), info() y describe(). Posteriormente podrán realizar consultas,
filtros y selecciones de columnas para responder preguntas sencillas
sobre el comportamiento de las ventas.

En las siguientes sesiones el mismo conjunto de datos servirá para
desarrollar procesos de limpieza, transformación, visualización y
análisis estadístico. Finalmente permitirá introducir conceptos de
inferencia estadística, comparación de grupos y formulación de hipótesis
a partir de evidencia obtenida de los datos.
